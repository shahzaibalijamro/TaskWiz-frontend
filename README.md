
# Task Management System – Frontend

Frontend application for the Task Management System, providing a responsive UI for user authentication and task management while communicating with the backend REST API. Built with the assistance of Google's Emergent AI.

**AI Tool Used:** https://emergent.sh


## Tech Stack

- React
- Javascript
- Vite
- Tailwind CSS
- Axios
- Netlify (For deployment)


## Features

- User authentication (login & signup)
- Protected routes based on authentication state
- Create, update, delete tasks
- Task status updates
- Responsive UI

```bash

  cd TaskWiz-frontend
  npm install
  npm run dev
```
